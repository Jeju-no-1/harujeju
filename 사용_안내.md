# 하루제주 (HARU JEJU) 웹사이트 v2

> 제주 농가와 상생하는 만덕푸드의 공식 브랜드 사이트
> **풀스크린 시네마틱 Hero · 한국어/영어 다국어 지원**

## 빠른 시작

### 1. 사이트 미리보기
- `index.html` 파일을 더블클릭하면 브라우저에서 바로 열립니다.
- Chrome / Edge / Safari / Firefox 모두 지원

### 2. 폴더 구조
```
harujeju/
├── index.html              # 메인 페이지 (시네마틱 Hero)
├── css/
│   └── style.css           # 전체 스타일
├── js/
│   ├── i18n.js             # 한/영 다국어 시스템
│   └── main.js             # 인터랙션 스크립트
├── pages/
│   ├── about.html          # 브랜드 스토리
│   ├── products.html       # 제품 목록
│   └── order.html          # 주문·문의·FAQ
├── images/
│   └── 사진_넣는_방법.txt  # 이미지 파일명 가이드
└── 사용_안내.md            # 이 문서
```

## v2 새 기능

### 풀스크린 시네마틱 Hero
- 화면 전체를 메인 사진으로 채우는 시네마틱 디자인
- 사진 위 흰색 큰 타이포그래피 + 우아한 단일 CTA
- 사진이 없을 때도 자동 그라데이션 폴백
- 부드러운 줌 애니메이션

### 한국어/영어 다국어
- 헤더 우측 **KO/EN** 토글로 즉시 전환
- 브라우저 언어 자동 감지
- 선택 언어 자동 저장 (다음 방문 시 유지)
- 모든 페이지 텍스트 번역 적용

### 무역 대응 디자인
- 영문 우선 헤더/푸터 (회사명, 주소, 연락처)
- 가격 KRW(₩) 표기
- 영문 보조 폰트 (Cormorant Garamond)
- 국제 표준 디자인 패턴

## 결제 시스템

본 사이트는 **네이버 스마트스토어 연동 방식**으로 결제를 처리합니다.

### 처리 흐름
1. 사이트에서 제품 카드 클릭
2. 자동으로 스마트스토어로 이동
3. 네이버페이/카드/계좌이체 등으로 결제
4. 구매 확정 후 사이트에서 리뷰 작성

### 카페24 연동 시
스마트스토어 URL을 카페24 쇼핑몰 URL로 일괄 치환하면 됩니다:
- `index.html`, `pages/products.html` 등에서 `https://smartstore.naver.com/mandukfood`를 카페24 URL로 변경
